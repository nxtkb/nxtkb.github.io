# Getting Started with Your NXTKB

Welcome to the next generation of typing comfort! Follow these steps to set up your new split keyboard.

## What's in the Box

When you receive your NXTKB, you'll find:
- Two keyboard halves (with USB-C charging cables)
- USB wireless receiver (if model includes wireless)
- Quick start guide
- Warranty information

## Initial Setup

### 1. Charging
- Charge both halves for at least 2 hours before first use
- Use provided USB-C cables with any standard USB power adapter
- Both halves can charge simultaneously

### 2. Pairing Wireless (If Applicable)
- Insert the USB wireless receiver into your computer
- Press the pairing button on each keyboard half
- Wait for the connection indicator light to turn solid green
- Test typing to confirm connection

### 3. Physical Positioning
- Position left and right halves approximately shoulder-width apart
- Ensure each half forms a 30Â° angle toward your body (or adjust to preference)
- Position in line with your natural arm extension
- Consider adjusting monitor positioning to match keyboard configuration

### 4. First Typing Experience
- It may feel unfamiliar at first - this is normal
- Start by typing simple words to build muscle memory
- Expect a learning period of 1-2 weeks for comfortable transition

## Basic Controls

### Layer Switching
- Fn + 1: Access function layer (special keys like F1-F12)
- Fn + 2: Navigation layer (arrow keys, home/end)
- Fn + 3: Adjust keyboard settings (if supported by model)

### Wireless Features
- Battery indicator via key light or separate status LED
- Auto-sleep mode after inactivity
- Manual connection controls (model dependent)

## Transition Tips

### Week 1: Familiarization
- Use for simple tasks initially
- Gradually increase typing complexity
- Don't worry about speed

### Week 2-3: Building Comfort
- Use as your primary keyboard
- Start to notice ergonomic benefits
- Practice reach positions for navigation

### Ongoing: Maximize Benefits
- Adjust settings based on comfort and preference
- Learn advanced features in the documentation
- Participate in the user community for tips and support

## Troubleshooting Start-up Issues

- If connection fails, ensure wireless receiver is properly seated
- If not charging, verify USB cable connections
- If keys are incorrect, check keyboard layout and language settings in operating system

Now you're ready to begin your ergonomic typing journey with NXTKB! Enjoy the health benefits and improved comfort your new keyboard will provide.
