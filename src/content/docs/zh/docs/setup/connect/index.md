---
title: "连接到设备"
---

NXTKB 键盘可以通过 USB 或蓝牙使用。新键盘第一次上手时，建议先用 USB 测试，因为这是确认左右两半供电正常、键盘能发送输入的最快方式。

## 开始之前

- 打开左右两半键盘电源。
- USB 连接和蓝牙输出都以左手半边作为主控。
- 右手半边也要保持开机，它会通过分体连接把输入发给左手。
- 需要确认浏览器实际收到哪些输入时，打开[键鼠测试](../keymap/input-tester/)。

## 先测试 USB

USB 是最可靠的第一次测试方式。

1. 打开左右两半电源。
2. 用 USB-C 线把左手半边连接到电脑。
3. 打开[键鼠测试](../keymap/input-tester/)。
4. 分别按左右两半的按键。
5. 如果你的键盘有鼠标层或触控板，也测试鼠标键、滚轮事件和指针移动。

如果测试页面能看到按键事件，说明键盘已经能正常发送输入。

## 蓝牙配对

确认 USB 正常后，再进行蓝牙配对。

1. 打开左右两半电源。
2. 如果你的键位中有 `BT_SEL 0` 到 `BT_SEL 4`，先选择要使用的蓝牙槽位。
3. 如果这个槽位以前配对过其它设备，用 `BT_CLR` 清除。
4. 打开电脑、手机或平板的蓝牙设置。
5. 找到键盘名称并配对。
6. 断开 USB，或使用键位中的输出切换键离开 USB 输出。
7. 再次打开[键鼠测试](../keymap/input-tester/)确认可以输入。

具体蓝牙控制键在哪一层取决于键位。找不到 `BT_SEL`、`BT_CLR` 或输出切换键时，先看对应型号的键位图：

- [Ferris Sweep Pro 键位](../keymap/ferris-sweep-pro-keymap/)
- [Ferris Sweep 键位](../keymap/ferris-sweep-keymap/)

## 重新连接或切换设备

ZMK 键盘可以保存多个蓝牙 profile。切换设备时：

1. 用 `BT_SEL` 选择目标蓝牙槽位。
2. 等几秒钟让键盘自动重连。
3. 如果不能重连，在目标设备的蓝牙列表中删除旧键盘记录，清除键盘上的对应槽位，然后重新配对。

USB 插着时通常会优先走 USB。想使用蓝牙时，断开 USB，或使用键位中的输出切换键。

## 排查

| 问题 | 建议 |
| :--- | :--- |
| USB 下完全不能输入 | 确认左右两半都已开机。换一根 USB-C 线，并把左手半边直接连到电脑。 |
| 左手能用，右手没反应 | 打开右手电源，然后重启左右两半。 |
| 蓝牙配对失败 | 先确认 USB 输入正常，用 `BT_CLR` 清除当前蓝牙槽位，在系统蓝牙列表删除旧记录后重新配对。 |
| 显示已配对但不能输入 | 检查是否仍插着 USB 或仍选中 USB 输出。重新选择蓝牙槽位并等待重连。 |
| 鼠标或触控板事件不清楚 | 使用[键鼠测试](../keymap/input-tester/)观察鼠标键、滚轮事件和指针移动。 |

如果 USB 正常但蓝牙反复失败，可以刷一次 `settings_reset` 清除 ZMK 保存的设置，再刷回正常固件。步骤见[如何刷写固件](../../firmware/how-to-flash-a-firmware/)。
