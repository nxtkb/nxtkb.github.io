- [Sweep Pro 配置文件](sweep-pro-configuration/)：说明哪些 `.keymap`、`config/*.conf` 和 `build.yaml` 文件控制 Sweep Pro 的细节行为。
- [Ferris Sweep 配置文件](ferris-sweep-configuration/)：说明 `cradio.keymap`、`cradio.conf`、behavior 和 `build.yaml` 如何控制 Ferris Sweep 的细节行为。
