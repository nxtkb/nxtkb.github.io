---
title: "Firmware"
---

NXTKB keyboards run ZMK firmware. This section covers the firmware workflows you need after changing keymaps, device settings, or keyboard configuration files.

- [How to Flash a Firmware](how-to-flash-a-firmware/) explains how to copy `.uf2` files onto the keyboard controller.
- [Build your own firmware](build-your-own-firmware/) explains how to build Ferris Sweep Pro and Ferris Sweep firmware with GitHub Actions or a local ZMK workspace.
- [Sweep Pro Configuration Files](sweep-pro-configuration/) explains which `.keymap`, `config/*.conf`, and `build.yaml` files control detailed Sweep Pro behavior.
- [Ferris Sweep Configuration Files](ferris-sweep-configuration/) explains which `cradio.keymap`, `cradio.conf`, behavior, and `build.yaml` files control detailed Ferris Sweep behavior.
